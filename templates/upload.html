How to use

{{error}}

Upload an image of a vegatable and let the vegtable recognizer recognize the vegatable.

The vegatable recognizer is trained on the vegatable image dataset, and can be found here . The dataset contains 15 vegatables:
  • Bean
  • Bitter Gourd
  • Bottle Gourd
  • Brinjal
  • Broccoli
  • Cabbage
  • Capsicum
  • Carrot
  • Cauliflower
  • Cucumber
  • Papaya
  • Potato
  • Pumpkin
  • Radish
  • Tomato

Optimal performance criteria

As this neural network is trained on a specific dataset, the performance of the model will be optimal if the uploaded images are similar to the dataset. As the dataset is not too diverse, the following critera are important:
  • The image should contain only one specie of vegatable from the list above
  • Image size should roughly be 224 x 224

    • If the uploaded image does not meet the criteria, it is likely that the model will predict the incorrect vegatable. In order to improve the model performance, a more diverse dataset must be added. More future improvements can be found on the future improvements page. {% endblock %}
